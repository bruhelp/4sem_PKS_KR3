# Симулятор работы лифтов в здании

## И что же это?

- Это консольное приложение на языке C++, представляющее из себя симуляцию работу нескольких лифтов в одном здании, где каждый из них обрабатывает вызовы пассажиров и изменяет своё состояние в зависимости от стратегии управления.

- Основной сутью проекта является многопоточное моделирование, при котором несколько лифтов функционируют одновременно.

---

## Архитектура проекта

Проект разделён на логические модули:

- core — основная логика лифтов
- scheduler — алгоритмы распределения вызовов
- simulation — управление симуляцией
- model — модели данных
- services — статистика, конфигурация, логирование
- result — результаты симуляции

---

## Алгоритмы

### FIFO

Вызовы обрабатываются в порядке поступления.

### Nearest Car

Выбирается ближайший лифт.

### Round Robin

Вызовы распределяются между лифтами по очереди.

### SCAN

Лифт движется в одном направлении, обрабатывая вызовы по пути.

## Как собрать?

```bash

cmake -S . -B build
# или
cmake -S . -B build -G "MinGW Makefiles"
cmake --build build

Запуск проекта

cd build
./ElevatorSimulator
На Windows:

cd build
ElevatorSimulator.exe

```

## Пример вывода

```bash

Time: 5

Elevator 1 Floor: 3 Load: 0
Queue: 5 7

Elevator 2 Floor: 8 Load: 0
Queue: 2
```
